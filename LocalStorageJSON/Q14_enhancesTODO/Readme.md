# ToDo Tasks Using LocalStorage

## Features
- Add a task
- Delete a task
- Mark task as completed or uncompleted (by clicking on the task text)
- Search tasks by typing text and clicking search button
- Tasks are saved in localStorage so they do not disappear after refresh

## How To Use
1. Type a task in the input field and click **Add Task**
2. The task will appear in the list below
3. Click on a task text to **mark it completed / uncompleted**
4. Click the **Delete** button to remove that task
5. To search:
   - Type a word in the search box
   - Click **Search** to see only matching tasks

## Technologies Used
- HTML
- JavaScript
- LocalStorage

## Notes
- Do not add empty tasks
- Search only filters tasks. It does not delete or change any task
- Completed tasks will show a line-through and green color
